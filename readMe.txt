Document de présentation du projet API_Clash_Royale

Dans ce projet nous interrongeons l' API officielle du jeu de Supercell Clash Royale afin
de connaître les 10 meilleurs joueurs par pays(Européens).

Donnéees :
- récupérer les données API via Php, les enregistrer en base de donnée
- interroger la bdd via Javascript getJson();

Objectif, interface graphique simple ou on sélection un pays dans une liste, et un tableau
contenant le player's ranking s'affiche et/ou si possible afficher les joueurs par pays
via une carte Google Api;
